# 半人马 CRM — AI Native 财税行业客户管理系统

面向财税行业的 AI Native CRM 系统。通过 AI 自动监控客户工商/税务/舆情变化，智能发现商机，生成沟通话术，让财税服务公司从被动响应变为主动经营。

## 核心功能

### 工作台
今日概览 — KPI 卡片、紧急待办、AI 商机推荐、政策动态、流失预警。一打开就知道今天要干嘛。

### AI 驾驶舱
全局大屏 — 客户状态分布(环形图)、任务完成率、营收排行(条形图)、最近动态时间线、风险预警、热门商机 TOP3、行业分布。老板看一眼就掌握全局。

### 客户管理
8 个模拟客户的完整 360 画像。支持搜索、状态过滤、负责人筛选、多维排序。客户详情页有三个 Tab：概览(服务+待办+商机)、动态(时间线)、预警(监控告警)。

### 智能日历
AI 自动生成的服务截止日日历。按状态/负责人筛选，按钮可直接推进任务状态(待处理→进行中→已完成)。

### 商机引擎
AI 基于客户画像自动发现的销售机会，含置信度评分和建议话术。支持状态推进(新发现→跟进中→已成交)。

### 政策雷达
实时监控财税政策变化，自动匹配受影响客户。展开可看 AI 建议行动和受影响客户列表。

### 智能监控
5 条 AI 监控规则(工商变更/税务风险/舆情/证照到期/法律风险)。每条规则可启停，下方是预警动态 Feed，支持按严重程度和处理状态筛选。

### AI 沟通助手
选择客户后，通过对话式界面让 AI 生成回访话术、社保调基通知、续费提醒、政策解读邮件等。AI 会根据客户画像定制内容。

### 客户导入
拖拽/粘贴任意格式数据，AI 自动识别列名和分隔符，智能映射到 CRM 字段，显示置信度。4 步流程：上传→解析→确认→完成。

## 技术栈

- React 19 + TypeScript
- Vite 8
- Tailwind CSS v4
- Lucide Icons
- 纯前端 Mock 数据 (无后端)

## 运行

```bash
npm install
npm run dev
# 默认 http://localhost:5173
```

## 模拟数据

- 8 个虚构浙江财税客户(杭州锐思科技、宁波海川贸易、浙江绿源环保等)
- 10 条待办任务
- 6 条 AI 商机推荐
- 4 条政策预警
- 12 条客户动态事件
- 5 条监控规则 + 7 条监控预警
- 客户导入字段映射示例

## 项目结构

```
src/
├── types.ts          # 全部类型定义
├── mock.ts           # Mock 数据
├── App.tsx           # 路由和布局
├── index.css         # Tailwind + 全局样式
├── main.tsx          # 入口
├── components/
│   └── Sidebar.tsx   # 侧边导航(分组+徽标)
└── pages/
    ├── Dashboard.tsx      # 工作台
    ├── Cockpit.tsx        # AI 驾驶舱
    ├── Customers.tsx      # 客户列表(搜索+筛选+排序)
    ├── CustomerDetail.tsx # 客户详情(概览/动态/预警)
    ├── Calendar.tsx       # 智能日历(状态可切换)
    ├── Opportunities.tsx  # 商机引擎(状态可推进)
    ├── Policies.tsx       # 政策雷达
    ├── Monitor.tsx        # 智能监控
    ├── Assistant.tsx      # AI 沟通助手
    └── Import.tsx         # 客户导入
```
